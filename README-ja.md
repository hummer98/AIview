# AIview

[English](README.md) | 日本語

macOS向けの高性能画像ビューア。大量の画像コレクション（1000枚以上）を即座に表示し、スムーズにナビゲーションできるよう最適化されています。

## 特徴

- **即時表示** - フォルダを開くとスキャン完了を待たずに最初の画像を表示
- **スムーズなナビゲーション** - 矢印キーの連打でもUIがブロックされない
- **AIプロンプト抽出** - PNGメタデータ（tEXtチャンク / XMP）からStable Diffusionのプロンプトを読み取り
- **プライバシーモード** - スペースキーで即座に全コンテンツを非表示
- **スマートプリフェッチ** - ナビゲーション方向に基づいて画像を先読み
- **サムネイルカルーセル** - 数千枚の画像でも仮想スクロールで高速表示
- **キーボード操作** - 削除、ナビゲーション、管理をマウスなしで実行
- **5段階お気に入り** - ★1〜★5で画像を評価、フォルダごとに永続保存
- **お気に入りフィルター** - 評価でフィルタリング、サブディレクトリも自動スキャン
- **スライドショー** - 設定可能な間隔（1〜60秒）で自動送り

## キーボードショートカット

### 基本ナビゲーション

| キー | 動作 |
|------|------|
| `←` / `→` | 前 / 次の画像へ移動 |
| `d` | 現在の画像を削除（ゴミ箱へ移動） |
| `i` | 画像情報パネルの表示切替（EXIF、プロンプト） |
| `t` | サムネイルカルーセルの表示切替 |
| `Space` | プライバシーモードの切替 |
| `s` | スライドショー設定ダイアログを開く |

### お気に入り

| キー | 動作 |
|------|------|
| `1` - `5` | お気に入り評価★1〜★5を設定（同じキーで解除） |
| `0` | お気に入り評価をクリア |

### フィルタリング（Shift併用）

| キー | 動作 |
|------|------|
| `Shift+1` | ★1以上でフィルター（サブディレクトリをスキャン） |
| `Shift+2` | ★2以上でフィルター（サブディレクトリをスキャン） |
| `Shift+3` | ★3以上でフィルター（サブディレクトリをスキャン） |
| `Shift+4` | ★4以上でフィルター（サブディレクトリをスキャン） |
| `Shift+5` | ★5以上でフィルター（サブディレクトリをスキャン） |
| `Shift+0` | フィルターをクリアしてサブディレクトリモードを終了 |

### スライドショー

| キー | 動作 |
|------|------|
| `↑` / `↓` | 間隔を1秒増加 / 減少 |
| `Space` | スライドショーを一時停止しプライバシーモードを有効化 |
| `Escape` | スライドショーを停止 |
| `←` / `→` | 手動ナビゲーション（タイマーをリセット） |

## 対応フォーマット

- JPEG (.jpg, .jpeg)
- PNG (.png)
- HEIC (.heic)
- WebP (.webp)
- GIF (.gif)

## 動作要件

- macOS 14.0（Sonoma）以降
- Xcode 15.0以降（ビルド時）

## インストール

### ソースからビルド

```bash
git clone https://github.com/hummer98/AIview.git
cd AIview
open AIview.xcodeproj
```

Xcodeでビルド＆実行（⌘R）。

### Task を使用する場合（オプション）

[Task](https://taskfile.dev/) がインストールされている場合：

```bash
task build
task run
```

## パフォーマンス設計

- **ストリーミングフォルダスキャン** - 全列挙を待たずに最初の画像を表示（バッチサイズ：50）
- **優先度ベースのプリフェッチキュー** - P0: 現在、P1: 隣接、P2: 近傍、P3: サムネイル
- **LRUメモリキャッシュ** - デコード済み画像をキャッシュして即座にアクセス（設定可能：128〜4096 MB）
- **ディスクサムネイルキャッシュ** - `.aiview/` サブフォルダに保存
- **タスクキャンセル** - 古いデコードタスクを即座にキャンセル
- **方向性プリフェッチ** - ナビゲーション方向に基づき、前方12枚、後方3枚を先読み

## データ保存

AIviewはフォルダごとのデータを隠し `.aiview/` ディレクトリに保存します：

```
your-image-folder/
├── image1.png
├── image2.jpg
└── .aiview/
    ├── favorites.json    # お気に入り評価
    └── thumbnails/       # キャッシュされたサムネイル
```
