# Implementation Plan

## Tasks

- [x] 1. スライドショータイマー機能の実装
- [x] 1.1 (P) タイマーサービスの実装
  - 指定間隔でコールバックを実行する機能を構築する
  - 一時停止・再開・停止・リセット制御を実装する
  - Swift Concurrencyを活用したキャンセル可能なタイマーを実現する
  - 同時に複数のタイマーが存在しないよう排他制御を行う
  - _Requirements: 2.1, 4.1, 4.2, 6.3_

- [x] 1.2 (P) タイマーサービスのユニットテスト
  - start/pause/resume/stop/resetの各動作を検証する
  - 間隔変更時の動作を検証する
  - キャンセル処理の正常性を確認する
  - _Requirements: 2.1, 4.1, 4.2, 6.3_

- [x] 2. 設定永続化機能の実装
- [x] 2.1 (P) 設定ストレージの拡張
  - スライドショー間隔をUserDefaultsに保存・読み込みする機能を追加する
  - デフォルト値（3秒）の登録処理を実装する
  - 1-60秒の範囲制約を適用する
  - _Requirements: 7.1, 7.2, 7.3_

- [x] 3. スライドショー状態管理の実装
- [x] 3.1 ViewModelへのスライドショー状態追加
  - アクティブ/一時停止/間隔の状態プロパティを追加する
  - 設定ダイアログ表示フラグを追加する
  - カルーセル表示状態の保存・復元用プロパティを追加する
  - タイマーサービスとの連携を実装する
  - _Requirements: 1.4, 2.1, 3.1, 3.2, 6.1, 8.1, 8.2_

- [x] 3.2 スライドショー制御メソッドの実装
  - 開始処理（設定取得、タイマー開始、カルーセル非表示）を実装する
  - 一時停止/再開のトグル処理を実装する
  - 終了処理（タイマー停止、カルーセル復元、リソース解放）を実装する
  - 手動ナビゲーション時のタイマーリセット処理を実装する
  - os.Loggerでスライドショー開始/終了/エラーをログ出力する（カテゴリ: slideshow）
  - _Requirements: 1.4, 3.1, 3.2, 4.1, 4.2, 6.1, 6.3, 6.4, 8.1, 8.2_

- [x] 3.3 間隔調整機能の実装
  - 上下矢印キーによる間隔増減処理を実装する（±1秒）
  - 最小1秒・最大60秒の範囲制限を適用する
  - 間隔変更時にタイマーへ新しい間隔を反映する
  - _Requirements: 5.1, 5.2_

- [x] 3.4 ループナビゲーションの実装
  - 最後の画像から最初の画像へ自動的に戻る処理を実装する
  - 最初の画像から前に移動した場合に最後の画像へ移動する処理を実装する
  - _Requirements: 2.4_

- [x] 3.5 スライドショー状態管理のユニットテスト
  - 状態遷移（開始→一時停止→再開→終了）を検証する
  - 間隔調整の範囲制限を検証する
  - ループナビゲーションの動作を検証する
  - カルーセル表示状態の保存・復元を検証する
  - _Requirements: 1.4, 2.4, 3.1, 3.2, 5.1, 5.2, 6.1, 8.1, 8.2_

- [x] 4. トースト通知機能の実装
- [x] 4.1 (P) トースト表示コンポーネントの作成
  - 画面下部中央に半透明の通知を表示する機能を実装する
  - 2秒後に自動で非表示になる動作を実装する
  - フェードイン/アウトアニメーションを適用する
  - 連続した通知を適切に処理する仕組みを実装する
  - _Requirements: 1.5, 3.3, 3.4, 5.3, 6.2_

- [x] 4.2 トースト状態管理の実装
  - 表示するメッセージの状態管理を実装する
  - 各スライドショーイベント時に適切なメッセージを設定する
  - 開始通知（表示間隔を含む）、一時停止/再開通知、終了通知、間隔変更通知を実装する
  - _Requirements: 1.5, 3.3, 3.4, 5.3, 6.2_

- [x] 5. 設定ダイアログの実装
- [x] 5.1 ダイアログUIの構築
  - 表示間隔を設定するスライダー（1-60秒、整数ステップ）を実装する
  - 開始・キャンセルボタンを配置する
  - キーボード操作のヘルプ情報を表示する
  - 保存済みの間隔を初期値として表示する
  - _Requirements: 1.1, 1.2, 1.3, 7.2, 8.3_

- [x] 5.2 ダイアログからの開始処理連携
  - 開始ボタン押下時に設定を保存してスライドショーを開始する
  - キャンセル時にダイアログを閉じる処理を実装する
  - 画像が0枚の場合は開始ボタンを無効化する
  - _Requirements: 1.4, 1.5, 7.1_

- [x] 6. キーボード操作の統合
- [x] 6.1 スライドショー開始キーの追加
  - Sキー押下時に設定ダイアログを表示する処理を追加する
  - 既存のキーボードハンドラに統合する
  - _Requirements: 1.1_

- [x] 6.2 スライドショー中のキー処理追加
  - スペースキーで一時停止/再開をトグルする処理を追加する
  - ESCキーでスライドショーを終了する処理を追加する
  - 左右矢印キーで画像を切り替えてタイマーをリセットする処理を追加する
  - 上下矢印キーで間隔を調整する処理を追加する
  - スライドショー中は上下矢印の既存動作を上書きする
  - _Requirements: 3.1, 3.2, 4.1, 4.2, 5.1, 5.2, 6.1_

- [x] 7. UI統合と表示調整
- [x] 7.1 メイン画面へのコンポーネント統合
  - 設定ダイアログをSheet表示として統合する
  - トーストオーバーレイを画像表示領域に重ねて配置する
  - スライドショー中にサムネイルカルーセルを非表示にする処理を統合する
  - _Requirements: 8.1, 8.2_

- [x] 7.2 フルスクリーン表示の調整
  - スライドショー中の画像表示がアスペクト比を維持することを確認する
  - ファイル情報オーバーレイがスライドショー中も表示されることを確認する
  - _Requirements: 2.2, 2.3_

- [x] 8. 結合テストと動作検証
- [x] 8.1 キーボード操作からのE2Eフロー検証
  - Sキーから開始、スペースで一時停止/再開、ESCで終了の一連の流れを検証する
  - 矢印キーによる手動ナビゲーションとタイマーリセットを検証する
  - 上下矢印による間隔調整を検証する
  - _Requirements: 1.1, 3.1, 3.2, 4.1, 4.2, 5.1, 5.2, 6.1_

- [x] 8.2 設定永続化の検証
  - 設定した間隔が次回起動時に復元されることを検証する
  - デフォルト値の適用を検証する
  - _Requirements: 7.1, 7.2, 7.3_

- [x] 8.3* UI表示の検証
  - トースト通知の表示タイミングと自動非表示を検証する
  - カルーセルの非表示/復元動作を検証する
  - 設定ダイアログのヘルプ情報表示を検証する
  - _Requirements: 1.5, 3.3, 3.4, 5.3, 6.2, 8.1, 8.2, 8.3_
