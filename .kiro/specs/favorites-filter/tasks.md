# Implementation Plan

## Tasks

- [x] 1. お気に入り情報の永続化基盤を構築する
- [x] 1.1 (P) お気に入りデータを管理するストアを実装する
  - フォルダごとのお気に入り情報（ファイル名→レベル1〜5のマッピング）をメモリ上で管理するactorを作成する
  - `.aiview/favorites.json`からのお気に入りデータ読み込み機能を実装する
  - ファイルが存在しない場合は空の辞書として初期化する
  - ファイル名をキー、お気に入りレベル（1〜5）を値とするJSON形式で保存する
  - `.aiview`フォルダが存在しない場合は自動作成する
  - 書き込みは`.atomic`オプションで安全に実行する
  - _Requirements: 2.1, 2.2, 2.3, 2.4_

- [x] 1.2 お気に入りストアのユニットテストを実装する
  - 読み込み・保存の基本動作テスト
  - ファイル未存在時の初期化テスト
  - レベル設定・解除のテスト
  - _Requirements: 2.1, 2.2, 2.3, 2.4_

- [x] 2. ViewModelにお気に入り操作機能を追加する
- [x] 2.1 フォルダオープン時のお気に入り読み込みを実装する
  - フォルダオープン処理にお気に入りストアからのデータ読み込みを統合する
  - 読み込んだお気に入り情報をViewModelのプロパティとして保持する
  - _Requirements: 2.2, 2.3_

- [x] 2.2 お気に入りレベルの設定・解除機能を実装する
  - 現在表示中の画像にお気に入りレベル（1〜5）を設定する機能を追加する
  - お気に入りレベル0で解除（お気に入りなし状態に戻す）する機能を追加する
  - 設定変更時に即座にストアへ保存する
  - 現在の画像のお気に入りレベルを取得するcomputed propertyを追加する
  - _Requirements: 1.1, 1.2, 1.4, 2.1_

- [x] 3. キーボード操作でお気に入りを設定できるようにする
- [x] 3.1 数字キー1〜5でお気に入りレベルを設定する
  - メインウィンドウのキー入力ハンドラに数字キー1〜5の処理を追加する
  - 画像表示中に数字キーを押すと対応するレベルが設定される
  - _Requirements: 1.1, 1.4_

- [x] 3.2 数字キー0でお気に入りを解除する
  - メインウィンドウのキー入力ハンドラに数字キー0の処理を追加する
  - 画像表示中に0キーを押すとお気に入りが解除される
  - _Requirements: 1.2_

- [x] 4. お気に入りレベルを視覚的に表示する
- [x] 4.1 (P) お気に入りインジケータViewを作成する
  - お気に入りレベル（1〜5）を星アイコンで表示するコンポーネントを作成する
  - メイン画像用（大サイズ）とサムネイル用（小サイズ）の2パターンに対応する
  - レベル0（未設定）の場合は非表示にする
  - _Requirements: 1.3_

- [x] 4.2 メイン画像表示にお気に入りインジケータを統合する
  - メイン画像の左上にお気に入りインジケータをオーバーレイ表示する
  - 現在の画像のお気に入りレベルに応じてインジケータが更新される
  - _Requirements: 1.3_

- [x] 4.3 サムネイルカルーセルにお気に入りインジケータを統合する
  - サムネイルの右下にお気に入りインジケータをバッジ表示する
  - 各サムネイルのお気に入りレベルに応じてインジケータが表示される
  - _Requirements: 1.3_

- [x] 5. フィルタリング機能を実装する
- [x] 5.1 ViewModelにフィルタリング状態管理を追加する
  - フィルタリングレベル（nil=無効、1〜5=有効）を管理するプロパティを追加する
  - フィルタリング条件に合致する画像インデックスの配列を保持する
  - フィルタリング有無、フィルタ後の画像数などのcomputed propertyを追加する
  - _Requirements: 3.3, 3.4, 4.2_

- [x] 5.2 指定レベル以上の画像をフィルタリングする機能を実装する
  - 指定したお気に入りレベル以上の画像のみを抽出するメソッドを追加する
  - フィルタリング時はフィルタ後の画像リストのみを表示対象とする
  - _Requirements: 3.1, 3.3, 3.4_

- [x] 5.3 フィルタリングを解除する機能を実装する
  - フィルタリングを解除して全画像を表示するメソッドを追加する
  - 解除時は元の全画像リストでの現在位置を維持する
  - _Requirements: 3.2, 5.4_

- [x] 6. フィルタリング時のナビゲーションを実装する

  - 左カーソルキーでフィルタ後リスト内の前の画像に移動する
  - 既存のナビゲーションメソッドをフィルタリング状態に対応させる
  - _Requirements: 3.5, 5.1, 5.2_

- [x] 6.2 フィルタリング時のプリフェッチを実装する
  - フィルタ後リストに基づいてプリフェッチ対象を更新する
  - フィルタリング変更時にプリフェッチウィンドウを再計算する

- [x] 7. SHIFT+数字キーでフィルタリングを操作できるようにする

- [x] 7.2 SHIFT+0でフィルタリングを解除する
