# Product Overview

AIview is a high-performance macOS image viewer designed for rapid browsing and selection of large image collections (1,000-2,000+ images). The primary goal is zero-latency image navigation where the first image appears instantly when opening a folder, and cursor key navigation remains responsive even during rapid key presses.

## Core Capabilities

- **Instant First-Image Display**: Shows the first image immediately upon folder selection, without waiting for full folder scan or thumbnail generation
- **Zero-Latency Navigation**: Predictive prefetching ensures next/previous images are ready before the user requests them
- **Rapid Image Selection**: Keyboard-driven workflow optimized for quickly reviewing, deleting, or inspecting AI-generated images
- **EXIF/Prompt Inspection**: Built-in metadata viewer for examining generation parameters embedded in AI-generated images

## Target Use Cases

- Reviewing large batches of AI-generated images (e.g., Stable Diffusion, Midjourney output)
- Quickly sorting and deleting unwanted images from bulk generation sessions
- Inspecting generation prompts and parameters stored in image metadata
- Browsing photography collections with minimal UI distraction

## Value Proposition

Unlike general-purpose image viewers that prioritize features over speed, AIview is laser-focused on responsiveness. The application maintains sub-100ms image transitions even with thousands of images, making it practical to review entire generation sessions without waiting.
